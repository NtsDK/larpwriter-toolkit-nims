NIMS MCP

Войдите учётной записью NIMS, чтобы получить токен для Cursor или другого MCP-клиента.

Пользователь: ()

Токен действует ч.

Cursor — добавьте в окружение и .cursor/mcp.json:

export NIMS_MCP_TOKEN="ваш-токен"

{
  "mcpServers": {
    "nims": {
      "url": "",
      "headers": { "Authorization": "Bearer ${env:NIMS_MCP_TOKEN}" }
    },
    "nims-readonly": {
      "url": "",
      "headers": { "Authorization": "Bearer ${env:NIMS_MCP_TOKEN}" }
    }
  }
}
